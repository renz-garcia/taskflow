# Project Overview

**TaskFlow** is a full-stack web application that allows users to create, track, and manage daily tasks efficiently. It's built using Django and SQLite for the backend and HTML, Tailwind CSS, and JavaScript for the frontend to practice CRUD operations, authentication, and responsive design.

# Core Features
- User registration and login
- Create, edit, delete tasks
- Mark tasks as complete/incomplete
- Display task list by completion status
